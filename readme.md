## How 2 Use
### Install

    $ pip install ngx

### Start

    $ ngx init <ngx-site-name-whatever-you-like>

### Config

    $ vim <ngx-site-name-whatever-you-like>/config.ini

### Run as daemon

    $ ngx start -d

### Enjoy

visit your ngx monitor site, login and enjoy:)
